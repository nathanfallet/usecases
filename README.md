## Installation

Add dependency to your `build.gradle` or `pom.xml`:

```groovy
compile 'me.nathanfallet.usecases:usecases:1.0.0'
```

```xml

<dependency>
    <groupId>me.nathanfallet.usecases</groupId>
    <artifactId>usecases</artifactId>
    <version>1.0.0</version>
</dependency>
```
